# Credit Card Transaction Dashboard

This repository contains a **Power BI dashboard** that analyzes credit card transaction data and customer information.

## Overview
The dashboard provides insights into:
- Credit card transaction patterns
- Customer spending behavior
- Transaction volume and trends
- Customer distribution

## Files Included
- `credit_card.csv` – Credit card transaction dataset  
- `customer.csv` – Customer information dataset  
- `cc_add.csv` – Additional credit card details  
- `cust_add.csv` – Additional customer details  
- `creditcard_CustomerReport.pdf` – Customer analysis report  
- `creditcard_TransactionReport.pdf` – Transaction analysis report  

## Tools Used
- **Power BI** for data visualization and dashboard creation  
- **CSV datasets** for transaction and customer data analysis

## Purpose
The goal of this project is to visualize and analyze credit card transaction data to identify trends, customer behavior, and key financial insights.
